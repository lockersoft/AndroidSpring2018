Room SQLite3 Abstraction Layer which handles all the database calls.

https://developer.android.com/training/data-storage/room/index.html

Add to build.gradle (project)
ext {
    roomVersion = '1.1.0-beta1'
}

Modify build.gradle  (app)
    implementation "android.arch.persistence.room:runtime:$rootProject.roomVersion"
    annotationProcessor "android.arch.persistence.room:compiler:$rootProject.roomVersion"
    androidTestImplementation "android.arch.persistence.room:testing:$rootProject.roomVersion"


    implementation "android.arch.persistence.room:runtime:1.1.0-beta1"
    annotationProcessor "android.arch.persistence.room:compiler:1.1.0-beta1"
